              REVERIE STUDIO v32 설치 안내

설치해 주셔서 감사합니다!

이 프로그램은 AI를 활용하여 유튜브 영상을 자동으로 제작합니다.


[ 설치 내용 ]

  - Reverie Studio 실행 파일
  - TTS 음성 모델 (약 1.4GB)
  - 배경음악 파일 (약 250MB)
  - 사용자 매뉴얼


[ 설치 요구사항 ]

  - Windows 10/11 64bit
  - 디스크 여유 공간: 최소 2.5GB
  - NVIDIA GPU (CUDA 지원 필수)


[ 추가 설치 필요 ]

  Reverie Studio를 사용하려면 다음 프로그램들이 별도로 필요합니다:

  1. Stable Diffusion WebUI (이미지 생성)
  2. GPT-SoVITS (음성 합성)
  3. Google Gemini API 키 (시나리오 생성)
  4. FFmpeg (영상 처리)

  자세한 설치 방법은 설치 후 "사용자 매뉴얼"을 참조하세요.

  라이센스 키는 구매 후 별도로 발급됩니다.
