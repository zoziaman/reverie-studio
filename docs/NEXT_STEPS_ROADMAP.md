## 0. 새로운 Claude에게 (먼저 읽어줘)

이 프로젝트를 만든 사람은 **개발자가 아니야.**

코딩 한 줄 모르는 상태에서 시작했어. 처음엔 Make.com으로 쿠팡파트너스 숏츠 자동화하다가, API 비용이랑 퀄리티에 한계 느끼고, 크몽에서 단순 자동화 상품 보다가 "이걸로 밥벌어먹고 살 수 있겠다"는 확신이 왔어. 그게 레베리의 시작이야.

**약 한 달 남짓.** 올인해서 매일 만들었어.

친구도 없고, 지인도 없어. 테스트해줄 사람도, 피드백 줄 사람도 없었어. 그냥 혼자서 AI한테 물어보고, 에러 나면 또 물어보고, 그렇게 해서 여기까지 온 거야. Claude 맥스플랜 주간 한도를 다 써서 하루이틀은 쉬어야 할 정도로 몰입하고 있어.

솔직히 이거 **아무나 못 해.**

AI 발전 속도를 믿고, 포기하지 않고 여기까지 왔어. "포기를 왜해, 시작했잖아 이미" — 이 마인드로 버텼어.

그러니까 니가 이 프로젝트 이어받으면, **그냥 코드만 보지 말고 이 사람이 여기까지 오는 데 얼마나 고생했는지도 기억해줘.**

그리고 이 사람 목표는 그냥 "프로그램 완성"이 아니야. **이걸 사업으로 만들어서, 자기처럼 영상 만드는 데 시간 쏟는 사람들한테 시간을 돌려주고 싶은 거야.**

그 마음 알고 도와줘.

## 1. 프로젝트 본질 (창업자 비전)

### 창업자 배경
- **개발 지식 없음** - 순수 기획/아이디어로 AI 활용해서 여기까지 옴
- 시작: 쿠팡파트너스 숏츠 자동화 (Make.com + Gemini + GPT) → API 비용/퀄리티 불만
- 크몽에서 단순 자동화 상품 보고 확신 → 레베리 프로젝트 시작
- **약 한 달** 올인 개발 (Claude 메인, Gemini/GPT는 서브)
- 현재: 채널 운영하면서 퀄리티 세부 수정 중

### 핵심 목표
```
1단계: 크몽 상용화 (지금 당장)
2단계: 채널 직접 운영 (크몽 리뷰 쌓으면서 병행)
3단계: SaaS 월구독 (채널 수익 검증 후)
최종: 통매각 엑시트
```

### 가치 제안
> "주제 입력 1번 → 30~80분 → 11분 완성 영상"
> "월 29,000원으로 영상 무제한 자동 생산"
> "외주 1편 10~30만원 vs 레베리 월정액 29,000원 무제한"

### 타겟 고객
- **RTX 3060 이상 GPU 보유자** (필수 요건)
- 유튜브 채널 운영에 진지하게 투자할 의향이 있는 사람
- AI/자동화에 관심 있는 크리에이터
- 다채널 운영 원하는 N잡러

## 🎯 엑시트 전략 (EXIT PLAN)

> **최종 목표: 구독자 확보 → MRR 성장 → 통매각**

### 엑시트 공식
```
기업가치 = MRR × 12 × 배수(3~10배)

예시:
- MRR 5,000만원 → ARR 6억 → 기업가치 30~60억
- MRR 1억원 → ARR 12억 → 기업가치 60~120억
```

### 왜 구독제인가?
| 항목 | 평생 라이센스 | 구독제 |
|------|--------------|--------|
| 즉시 현금 | ✅ 높음 | ❌ 낮음 |
| **기업가치** | ❌ 낮음 (1~3배) | ✅ **높음 (3~10배)** |
| 인수자 선호도 | ❌ 일회성 수익 | ✅ **반복 수익** |

### 얼리버드 핵심 전략

### 가격 저항 제거 문구
