# 훅 장면 시스템

> TV 스타일 타이포그래피 훅
> 최종 업데이트: 2026-02-03 (v57.6.5)

---

## 개요

v57.6.5에서 기존 훅 장면(TTS + SD 이미지)을 **TV 스타일 타이포그래피**로 변경.

---

## 변경 전후 비교

### 기존 (v57.6.4 이전)
- 후킹 대본으로 TTS 생성
- SD WebUI로 이미지 생성
- 자막 오버레이

### 현재 (v57.6.5)
- 검은 배경 + 채널 레이블 + 주제 텍스트
- TTS/이미지 생성 없음 (빠름)
- 채널별 스타일링

---
